# Gulp file project
Example gulp project containing the following features

## HTML
- Minification

## Styles
- SASS compilation
- Linting
- Autoprefixing
- Formatting
- Minification

## Scripts
- Linting
- Concatenation
- Minification

## Images
- Compression

## Server
- Local webserver with LiveReload

## Other features
- Distinct development and production builds
- Separation of tasks into multiple files
